About

The purpose of this website is to find out which pets are present at the fictional rescue organisation called Adoptr.

This is done by the user searching through the database using the form to find out whether or not their desired pet is there and up for adoption.

The database then sends back the information based on the search query and it displays as a table at the bottom of the form, showcasing the pet name, breed, colour, coat, size, sex, age and whether or not they are cat, dog or child friendly if they are up for adoption.


This website has been created using a MEAN stack (MongoDB, Express, Angular, NodeJs) for the backend and HTML5, CSS3, JavaScript and Bootstrap for the front end.
